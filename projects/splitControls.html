-= Split Controls =-

 Split Controls has been a progressively developed personal project of mine. First as an self-driven introduction to computer networking, it has since become a staple social event for friends of mine and myself.

 The program is a locally hosted web page that enables inputs from participants' computers to be replicated on the hosting device. Matched with custom weighted bindings for various pre-existing games, The system allows multiple people with fragments of a single-player game's controls to have a unique collaborative experience.
